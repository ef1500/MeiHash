# MeiHash - Dual-Mixing Non-Cryptographic 128-Bit Hash

</div>

---

MeiHash is a non-cryptographic hash function, whose inspiration came from none other than Mei Aihara, the lead heroine of an anime.

MeiHash achieves about 10.3 GB/s and passes all smhasher tests. It uses two paths to create high-entropy.
